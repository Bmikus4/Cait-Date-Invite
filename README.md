# For Cait

An interactive date invitation. A wax-sealed letter that opens to reveal the plan.

- **When:** Saturday, June 27th, 2026, 4:00pm sharp
- **Where:** Ascend Climbing, South Side, Pittsburgh (2141 Mary St)

Single self-contained `index.html`. No build step. Deploys to Vercel as a static page.

Aesthetic borrowed from the House of Hud stationery (Georgia + Cormorant, cream paper,
gold `#c8a87e`, hairline rules); fully re-architected around the open-the-letter moment.
Mobile-first, animated, respects `prefers-reduced-motion`.
